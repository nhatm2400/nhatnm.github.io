### Week 3 Objectives:

* Get familiar with deploying and managing relational databases on Amazon RDS.  
* Practice deploying applications with EC2 Auto Scaling Group and Elastic Load Balancing.  
* Use Amazon CloudWatch to monitor the system (Metrics, Logs, Alarms) and practice connecting DNS Hybrid.  
* Master operations with AWS CLI, from installation to initializing basic services (EC2, S3, IAM, VPC, …).  
* Learn and practice the main features of Amazon DynamoDB.
