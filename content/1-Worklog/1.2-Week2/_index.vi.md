### Kết quả đạt được tuần 2:

* Hiểu rõ hơn về các dịch vụ cốt lõi của AWS, đặc biệt:
  * Amazon VPC (Virtual Private Cloud)
  * Amazon EC2 (Elastic Compute Cloud)
  * AWS Cloud9
  * Amazon S3
  * IAM Roles và quản lý truy cập

* Hoàn thành các thực hành:
  * Tạo và cấu hình Amazon VPC và Subnet  
  * Khởi tạo EC2 Instance chạy Windows và Linux  
  * Triển khai ứng dụng Node.js trên EC2 (Windows & Linux)  
  * Triển khai ứng dụng AWS User Management trên EC2 chạy Windows Server 2022  
  * Tạo và cấu hình IAM Roles cho EC2  
  * Sử dụng AWS Cloud9 để phát triển, chạy lệnh CLI và chỉnh sửa file trực tiếp trên môi trường đám mây  
  * Triển khai website tĩnh trên Amazon S3, bao gồm cấu hình bucket, bật Static Website Hosting và quản lý quyền truy cập object  

* Biết cách kết hợp AWS Management Console và AWS CLI để quản lý tài nguyên hiệu quả.

* Tham gia họp nhóm để thảo luận chiến lược, ý tưởng dự án và phân chia công việc.
